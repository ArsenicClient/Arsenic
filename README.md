# Arsenic

Arsenic is a 1.8.9 Minecraft cheat that specifically focuses on hypixel / grim anticheat. Any features that flag grim should be removed or recoded.
Arsenic is a forge mod.

The Client Features:
- Module System
- Command System
- Custom Events 
- Mixins
- Config System
- Custom Gui with Themes

Currently there are not too many modules, however, it is in a useable state and most should bypass fine

## How To Build/Edit

- clone repo
- run setup.bat
- open Arsenic.ipr
- load gradle project
- skid away :D
